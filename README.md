## About The Project

This is a Flutter-based mobile application for Community Health Workers (CHW) to manage their tasks and interact with the community. The app is designed to help CHWs keep track of their patients, appointments, and other important information.

## Features

*   **Authentication:** Secure user authentication using Firebase Authentication.
*   **Database:** Cloud Firestore for storing and retrieving data in real-time.
*   **Storage:** Firebase Storage for storing files and images.
*   **Real-time Updates:** Firebase Realtime Database for real-time data synchronization.
*   **Beautiful UI:** A clean and intuitive user interface built with Flutter and Google Fonts.

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

*   Flutter SDK: [https://flutter.dev/docs/get-started/install](https://flutter.dev/docs/get-started/install)
*   Firebase CLI: [https://firebase.google.com/docs/cli](https://firebase.google.com/docs/cli)

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Mahad-Ghauri/Community-Health-Worker--User.git
    ```
2.  Install packages
    ```sh
    flutter pub get
    ```
3.  Run the app
    ```sh
    flutter run
    ```
